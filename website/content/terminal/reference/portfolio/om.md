Provides omega ratio of the selected portfolio.

### Usage

```python
om [-s START] [-e END]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| start | Start of the omega ratio threshold | 0 | True | None |
| end | End of the omega ratio threshold | 1.5 | True | None |
