### Usage

```python
demark [-m MIN_TO_SHOW]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| min_to_show | Minimum value of indicator to show (declutters plot). | 5 | True | None |
