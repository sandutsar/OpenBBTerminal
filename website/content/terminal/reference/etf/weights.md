Look at ETF sector holdings

### Usage

```python
weights [-m MIN] [--raw]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| min | Minimum positive float to display sector | 5 | True | None |
| raw | Only output raw data | False | True | None |
