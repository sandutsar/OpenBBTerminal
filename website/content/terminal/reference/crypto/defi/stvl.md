Displays historical values of the total sum of TVLs from all listed dApps. [Source: https://docs.llama.fi/api]

### Usage

```python
stvl [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of records to display | 10 | True | None |
