Displays terra blockchain staking ratio history. [Source: https://fcd.terra.dev/swagger]

### Usage

```python
sratio [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of days to show | 90 | True | None |
