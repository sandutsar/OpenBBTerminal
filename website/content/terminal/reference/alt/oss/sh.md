### Usage

```python

```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| repo | Repository to search for star history. Format: org/repo, e.g., openbb-finance/openbbterminal | None | False | None |
