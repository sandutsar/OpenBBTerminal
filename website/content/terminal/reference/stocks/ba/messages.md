Print up to 30 of the last messages on the board. [Source: Stocktwits]

### Usage

```python
messages [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | limit messages shown. | 30 | True | None |
