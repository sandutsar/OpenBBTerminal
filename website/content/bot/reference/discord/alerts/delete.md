This command allows the user to delete an active alert they have set.

### Usage

```python wordwrap
/alerts delete alert
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| alert | Alert to delete | False | None |

```
/alerts delete alert:
```
