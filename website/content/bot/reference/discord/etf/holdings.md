This command returns the holdings for an ETF sorted by percent of ETF.

### Usage

```python wordwrap
/etf holdings [etf]
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| etf | ETF ticker (only for `holdings`) | True | None |


---

## Examples

```
/etf holdings SPY
```
