This command allows the user to display customers of the company using the given symbol.

### Usage

```python wordwrap
/dd customer ticker
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker | False | None |


---

## Examples

```
/dd customer ticker:AMD
```
