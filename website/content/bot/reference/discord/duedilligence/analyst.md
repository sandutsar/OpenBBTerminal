This command allows the user to retrieve analyst recommendations for a given stock ticker.

### Usage

```python wordwrap
/dd analyst ticker
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker | False | None |


---

## Examples

```
/dd analyst ticker:AMD
```
