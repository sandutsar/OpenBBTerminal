This command allows the user to retrieve futures prices on the stock market.

### Usage

```python wordwrap
/futures [all_futures]
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| all_futures | Display all futures. Default False | True | None |


---

## Examples

```
/futures
```
