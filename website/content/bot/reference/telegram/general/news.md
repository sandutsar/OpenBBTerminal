This command allows the user to retrieve the latest news for a given ticker.

### Usage

```python wordwrap
/news ticker
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker | False | None |


---

## Examples

```
/news AMD
```
