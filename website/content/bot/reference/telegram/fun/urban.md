This command allows the user to search for a definition of a word on Urban Dictionary.

### Usage

```python wordwrap
/urban word
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| word | The word or words to search for. | False | None |


---

## Examples

```
/urban Shill
```
