This command allows the user to retrieve recent house trades for a specific ticker.

### Usage

```python wordwrap
/house [ticker]
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker (optional) | True | None |


---

## Examples

```
/house GE
```
