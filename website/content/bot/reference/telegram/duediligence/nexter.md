This command will retrieve the next earnings date for the given stock ticker symbol.

### Usage

```python wordwrap
/nexter ticker
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker | False | None |


---

## Examples

```
/nexter AMD
```
