This command allows users to retrieve the list of suppliers associated with the company with the given ticker.

### Usage

```python wordwrap
/supplier ticker
```

---

## Parameters

| Name | Description | Optional | Choices |
| ---- | ----------- | -------- | ------- |
| ticker | Stock Ticker | False | None |


---

## Examples

```
/supplier AMD
```
